# Funcionalidad de Cálculo IMC

Esta rama contiene la documentación relacionada con el módulo de cálculo del índice de masa corporal.

El sistema permite ingresar los datos necesarios del usuario para realizar el cálculo del IMC,
mostrar el resultado obtenido y asociarlo con el historial correspondiente.

Componentes relacionados:
- IMC.java
- IMCServlet.java
- calcularIMC.jsp
- HistorialIMC.java
