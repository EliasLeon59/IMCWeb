# Funcionalidad de Historial

Esta rama contiene la documentación relacionada con el módulo de historial de cálculos IMC.

El sistema permite consultar los registros de cálculos realizados por los usuarios,
mostrando la información almacenada previamente en la base de datos.

Componentes relacionados:
- HistorialIMC.java
- HistorialDAO.java
- HistorialREST.java
- historial.jsp
