# Funcionalidad REST

Esta rama contiene la documentación relacionada con los servicios REST de la aplicación IMCWeb.

El sistema implementa recursos REST para la consulta y manejo de información del historial,
permitiendo la comunicación mediante representaciones de datos compatibles con servicios web.

Componentes relacionados:
- HistorialREST.java
- JakartaRestConfiguration.java
- JakartaEE11Resource.java
- Representaciones JSON y XML
